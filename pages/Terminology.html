专业术语解释

这个页面用于解释一些你可能会看见的专业术语，但是在实际的lua里你不必声明这个变量的类型，lua会自动识别。

这是常见的数据类型

数据类型

  • String -字符串
  • Bool -布尔类型
  • Int -整数类型
  • Float -浮点数
  • Array -数组类型

String

数据类型: String

这个类型叫字符串，如果有个变量被声明为字符串，那么这个变量的值可以是任意字符组成而成，但是这个变量的值必须要用引号引起来。

举个例子:

  • hello:String = "abcd1234./?" - 这个变量的值被赋"abcd1234./?"的值

错误示范:

  • hello:String = abcd1234./? - 我们提到过这个变量的值必须要用引号引起来，前提是abcd1234./? 不是一个变量。

Bool

数据类型: Bool

这个类型叫布尔类型，他只能表示True和False，代表“真”和“假”，如果有个变量被声明为布尔值，那么这个变量值只能为“True”或“False”。

举个例子:

  • hello:Bool = true - 这个变量的值被赋真（true）值
  • hello:Bool = false - 这个变量的值被赋假（false）值

错误示范:

  • hello:Bool = "true" - 布尔变量的真假值不需要引号。

Int

数据类型: Int

这个类型叫整数类型，顾名思义，如果一个变量被声明为整数类型，那么这个变量值只能是整数，不能包含字母，符号（小数点也算符号）等不是数字的字符。

举个例子:

  • hello:Int = 114514 - 这个变量的值被赋值为114515
  • hello:Int = 1234 - 这个变量的值被赋值为1234

错误示范:

  • hello:Int = "114514" - 整数变量的值不需要引号。
  • hello:Int = 11451.4 - 整数变量的值不能有符号（小数点）。

Float

数据类型: Float

这个类型叫浮点数，浮点数其实和整数类型的区别就是你可以使用小数点（只能是小数点，其他符号都不行，小数点的位置要在哪里打你是知道的），如果一个变量被声明为浮点数，那么他既可以是整数，也可以是小数，和整数类型一样，不能包含字母和除小数点外的符号。

举个例子:

  • hello:FLOAT = 114514.0 - 这个变量的值被赋值为114515.0
  • hello:FLOAT = 114514 - 前面提到过他可以是整数。
  • hello:FLOAT = 1234.567 - 这个变量的值被赋值为1234.567

错误示范:

  • hello:FLOAT = "114514.0" - 浮点数的值不需要引号。
  • hello:FLOAT = 114514. - 如果你会这么打小数点我能笑你一辈子。
  • hello:FLOAT = .114514 - 我能笑你十辈子。
  • hello:FLOAT = 11451。4 - 我能笑你百辈子。

Array

数据类型: Array

这个类型叫数组类型，我们通常称为组，他可以用一个变量表达多种不同的值，组里面的各个数值我们统称为"成员"，他们的先后顺序是从'0'开始排序，从左往右数的，他们的可以用索引（就是两个中括号里一个数字，如hello[2]，就索引了数组里第三个成员的值）获取对应值。

举个例子:

  • hello:Array = [abc,123,114514] - 这个组变量的值被赋值为[abc,123,114514] ，hello[0]就表示这个组里的第一个值，所以hello[0]的值就是"abc"，以此类推hello[1]就是"123"，hello[2]就是"114514"。

错误示范:

  • hello:Array = 114514,123,www - 数组必须要用中括号括起来。
  • hello:Array = [114514,123,www]，你使用了hello[-1]或索引了一个小于0的数或索引了一个非整数。 - 这样做会出现不可预料的结果，所以我不建议你这样搞。
  • hello:Array = [114514,123,www]，你使用了hello0尝试索引 - 别忘了索引数要在中括号里。
